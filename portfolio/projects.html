Project 1

React JS & Solidity- BlockChain Cloud Storage

Details

A website created with React JS, truffle, Ganache, Ethereum & IPFS to store files into the off-chain cloud storage. User can log in and upload their files into the IPFS off-chain.

Project 2

Laravel - Classroom Website

Details

A simple website with Laravel , Blade , HTML & CSS to create new courses, add course materials and YouTube links with a minimal dashboard for the teachers to add, update new courses. Courses can be viewed from courses tab. Also includes the feature to add different course metarials of different universities.





The challenges I faced:
  • Database management, as this website was built to support a large number of students.
  • An efficient hosting provider; for now, it is hosted on a free site.
Project 3

Ruby on Rails - Search Engine

Details

A AI search engine using Elastisearch and searchkick to search Law reports, judgements and rulings. Project uses redis for caching.





The challenges I faced:
  • As Elasticsearch was not free, I had to find a free hosting for elasticsearch.
  • A Large database was needed to store out Law documents
  • Another hosting was needed for rails and redis.
Project 4

PHP HTML CSS & JS - Social Media blogging site

Details

A minimal website with Raw PHP, JS, HTML & CSS to create blogposts, upload pictures and react to others posts, User can use the real-time search capabilities to find any posts easily. And if the user gets bored they can use the Chat AI - Open AI - API for ChatGPT.





The challenges I faced:
  • OpenAI API was difficult to use, as they have changed their API endpoints.
  • GPT-3 was not available anymore. We had to use GPT-3.5
