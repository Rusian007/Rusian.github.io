React JS & Solidity- BlockChain Cloud Storage

A website created with React JS, truffle, Ganache, Ethereum & IPFS to store files into the off-chain cloud storage. User can log in and upload their files into the IPFS off-chain.

The challenges I faced making this website:

  • Solidity was difficult to learn.
  • IPFS access was limited for public url so we had to find a alternative.
Laravel - Classroom Website

A simple website to create new courses, add course materials and YouTube links with a minimal dashboard for the teachers to add, update new courses. Courses can be viewed from courses tab. Also includes the feature to add different course metarials of different universities.

The challenges I faced making this website:

  • Database management, as this website was built to support a large number of students.
  • An efficient hosting provider; for now, it is hosted on a free site.
Ruby on Rails - Search Engine

A AI search engine using Elastisearch and searchkick to search Law reports, judgements and rulings. Project uses redis for caching.

The challenges I faced making this website:

  • As Elasticsearch was not free, I had to find a free hosting for elasticsearch.
  • A Large database was needed to store out Law documents
  • Another hosting was needed for rails and redis.
